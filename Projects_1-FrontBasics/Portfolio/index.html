Projetos

Projeto 1

Meu primeiro projeto, consiste numa réplica simplificada do Spotify, contendo uma homepage e rotas dedicadas para artistas e músicas. Nesse sentido, o desenvolvimento desse web app foi essencial para consolidar conceitos fundamentais do desenvolvimento full stack com JavaScript. Além disso, o processo de deploy trouxe a necessidade de adaptar o código para rodar na nuvem. Isso envolveu testes no terminal Ubuntu, configuração do backend no Render e hospedagem do frontend na Vercel. Tudo isso, para não a ver mais a necessidade de usar as dependências locais e tornar o projeto acessível online.
